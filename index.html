Las hojas de estilo en cascada ( CSS ) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML o XML (incluidos dialectos XML como SVG , MathML o XHTML ). CSS describe cómo se deben representar los elementos en la pantalla, en papel, en voz o en otros medios.
